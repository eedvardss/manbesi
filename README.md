# Manbesi — Landing Page

Two design variants for [manbesi.lv](https://manbesi.lv). Dark theme, React + Vite.

## Setup

```bash
npm install
npm run dev
```

## Viewing the variants

Once the dev server is running, open your browser:

| Variant | URL | Description |
|---------|-----|-------------|
| **Version 1** | http://localhost:5173 | Centered single-column layout |
| **Version 2** | http://localhost:5173/v2.html | Split layout — content left, form right |

## Requirements

- Node.js v18+
- npm v9+
